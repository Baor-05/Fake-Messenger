Tính năng

Trải nghiệm Messenger Desktop gọn, nhanh và tách biệt tài khoản.

Đa tài khoản

Mỗi tài khoản dùng partition riêng cho cookies, cache và localStorage.

Quyền riêng tư

Hỗ trợ chặn trạng thái đã xem và đang nhập.

Gọi điện

Popup gọi điện/video call chạy trực tiếp trong Electron với quyền mic/camera.

System Tray

Thu nhỏ xuống khay hệ thống, mở lại nhanh bằng phím tắt.

Cài đặt

01

Cài dependencies

Chạy npm install.

02

Chạy app

Chạy npm start.

03

Build

Chạy npm run build để tạo file cài đặt.

Phím tắt

Ctrl+Shift+M

Mở hoặc ẩn cửa sổ Messenger.

FAQ

Dữ liệu đăng nhập được lưu ở đâu?

Dữ liệu session được lưu cục bộ theo profile trên máy đang chạy app.

App có hỗ trợ gọi điện không?

Có. Cuộc gọi Messenger mở trong cửa sổ Electron riêng và dùng quyền microphone/camera của hệ thống.
